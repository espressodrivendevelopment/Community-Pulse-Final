# Community Pulse 🌍

Empowering local change through grounded insights.

## 🔗 Live App
👉 [Click here to use Community Pulse](https://community-pulse-final.web.app)

## 📱 Features
- Submit community needs (AI-powered by Gemini)
- Real-time needs map
- Volunteer directory & registration
- Smart task matcher
- Impact dashboard with AI insights

## 🛠️ Built With
- Google Stitch (UI Design)
- Firebase Firestore (Database)
- Firebase Hosting (Deployment)
- Gemini 2.0 Flash (AI Analysis)

## 🚀 How to Use
1. Visit the live app link above
2. Submit a community need
3. Register as a volunteer
4. Match volunteers to needs
5. Track impact on the dashboard
